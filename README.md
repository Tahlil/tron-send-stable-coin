# tron-send-stable-coin

Sent USDT and USDC for Tron Mainnet and Testnets.

## How to run

- First, install the dependencies using `npm`

```bash
    npm i
```

- Then create the `.env` from `.env.example` and update it

```bash
    cp .env.example .env
```

- Run the code using the following command

```bash
    node index.js
```
